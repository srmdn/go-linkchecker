# Security Policy

## Reporting

Report vulnerabilities privately. Do not open public issues for active vulnerabilities.

- GitHub private security advisory: https://github.com/srmdn/go-linkchecker/security/advisories/new
- Email: mail@saidwp.com

Include:
- Affected versions or commits
- Reproduction steps
- Impact assessment
- Suggested mitigation

## Response Targets

- Acknowledge within 72 hours
- Initial triage within 7 days
- Remediation plan after validation

## Coordinated Disclosure

Please wait for a fix or an agreed mitigation window before public disclosure.

## Supported Versions

Only the latest release is actively maintained.
